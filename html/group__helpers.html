Detailed Description

Helper functions for Drush Rush commands.

These helper functions include functions to:

  • provide wrappers to Drush functions and commands
  • provide error and message formatting
  • provide service functions for Rush commands

Though not strictly internal or private, these functions are mainly used by other Rush functions, and are not structured like operations functions which may be called directly by other code with parameters passed at call time.
