Detailed Description

Site operation functions for Drush Rush.

Includes functions to:

  • Run drush make
  • Run drush site-install
  • Derive and set site-install parameters
  • Open a site in a browser

rush_op_set_drupal_variables (   $params)

Wrapper for drush variable-set.

This rush command takes an array of Drupal variable names and their values and sets them with "drush variable-set"

Parameters
array$paramsAn associative array of all rush parameters.
  • $params['build']['site_alias_record'] array Required. A drush site alias record.
  • $params['drupal_variables'] array Required. A flat array with variable names as keys and their values as the array values. For example: $params['drupal_variables'] = array( 'site_name' => 'My Great Blog Site', 'clean_url' => '1', );
Returns
bool TRUE on success. FALSE with errors.
