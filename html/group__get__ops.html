rush_get_operations (   $params)

Parse rush operations files and return an array of operations.

The following is the order of parsing:

  1. Get rush operations from the top job rush ini file if it exists.
  2. Get rush operations from the top job operations php file if it exists.
    • Operations in the top job php file will run after all other operations to this point.
  3. Get rush operations from the job rush ini file if it exists.
  4. Get rush operations from the job operations php file if it exists.
    • Operations in the job php file will run after all other operations to this point.
  5. Get pre rush operations from the environment rush ini file if it exists.
  6. Get pre rush operations from the environment operations php file if it exists.
Parameters
array$paramsAn associative array of rush parameters. The params are not directly used in the function, however passing them to the function makes them available to the include files.
Returns
mixed An associative array of all rush operations, with any enironment ops prepended. False via drush set error if now operations are found.
Backlog:
Fix to just return an empty array().
